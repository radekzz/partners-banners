Show partners logo in widget area.

== Description ==

Show partners logos in widget area. User can click on logo to open partner URL.

== Installation ==

1. Upload `partners-banners` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add new partners in `Partner banners`
4. Add widget `Partners Banners` to your widget area

== Frequently Asked Questions ==

How big are partner images? 
Partner images have 100% width and auto height, but you can style them in css.

How can i add the alt tag to images?
Title of Partner post is alt tag.
